# 🚗 Sistema de Alquiler de Carros en Java

Aplicación que permite gestionar el alquiler de vehículos, registrando clientes y controlando los alquileres realizados.

## 🚀 Funcionalidades
- Registro de clientes
- Alquiler de vehículos
- Cálculo del costo según días de uso
- Devolución de vehículos
- Validación de cliente en el sistema

## 🧠 Lógica implementada
Se implementa el uso de listas para almacenar clientes y alquileres, además de validaciones para garantizar que el vehículo sea devuelto por el cliente correcto.

## 🛠️ Tecnologías
- Java
- Listas (ArrayList)
- Programación Orientada a Objetos

## 📂 Estructura
- Clase Cliente
- Clase Alquiler
- Menú interactivo en consola
